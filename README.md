# Story Book Generator

一个用于 **稳定生成儿童绘本 Prompt** 的极简工具。\
目标不是工程化，而是让创作可以 **持续、可复现、低阻力**。

------------------------------------------------------------------------

## 🚀 项目解决了什么问题？

在使用 AI（如 Gemini Storybook）创作绘本时，常见痛点：

-   世界观 / 风格 / 故事容易混在一起\
-   Prompt 手动拼接容易遗漏或顺序错误\
-   历史生成结果难以复现\
-   修改规则后，不知道是哪一版生成的图片

本项目提供一个 **可复现的 Prompt 构建流程**。

------------------------------------------------------------------------

## ✅ 核心能力

### 1️⃣ 结构化 Prompt 自动生成

系统会自动拼接：

-   IMAGE RULES（硬视觉规则）
-   WORLD（世界观）
-   STYLE（美术规范）
-   STORY（故事内容）

避免手动复制导致的错误。

------------------------------------------------------------------------

### 2️⃣ 强可复现性

每个生成的 Prompt 顶部都会写入：

-   Git commit
-   rules 最新 commit
-   story 文件路径
-   构建时间

未来可以精准回溯：

> "这本绘本，是用哪一版规则生成的？"

------------------------------------------------------------------------

### 3️⃣ 极低使用成本

日常只需要两步：

    写故事 → build prompt → 粘贴到 Storybook

没有复杂流程。

------------------------------------------------------------------------

## 📁 项目结构

    story_book/
    │
    ├── rules/        # 世界观 & 风格规则（长期稳定）
    ├── stories/      # 每本书的故事
    ├── prompts/      # 自动生成的 Prompt（建议提交到 git）
    ├── builder/      # Prompt 模板
    └── build_prompt.sh

------------------------------------------------------------------------

## 🧭 如何使用？

### Step 1 --- 写故事

创建文件：

    stories/my_story.md

只写故事本身。

不要在这里写风格或世界观。

------------------------------------------------------------------------

### Step 2 --- 构建 Prompt

在项目根目录执行：

    ./build_prompt.sh stories/my_story.md

生成：

    prompts/my_story_prompt.md

直接复制到 Gemini Storybook。

------------------------------------------------------------------------

## ⭐ 推荐工作流

### ✅ 尽量保持稳定

不要频繁修改：

    rules/image_core.md

这是视觉一致性的关键。

世界观可以演进，但视觉规则建议保持稳定。

------------------------------------------------------------------------

### ✅ 提交 prompts

建议把 `prompts/` 一起提交到 Git。

这样未来可以：

-   回溯生成历史
-   对比不同规则效果
-   快速复现某一本绘本

------------------------------------------------------------------------

## ⚠️ 编码建议（非常重要）

**所有文件必须使用 UTF-8。**

不要从：

-   Word\
-   飞书\
-   石墨\
-   Notion

直接导出 Markdown。

推荐编辑器：

-   VSCode\
-   Cursor\
-   Sublime

------------------------------------------------------------------------
