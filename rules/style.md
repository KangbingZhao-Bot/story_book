# 《咚咚车库城》视觉与风格规范（v2）

目标：在不照搬原绘本画风的前提下，保持孩子认知一致性，避免世界观混淆；全书画风一致、车辆识别一致。

## 一、核心识别层（必须一致）

### 1️⃣ 车辆颜色固定（第一优先级）
- 翻翻（翻斗车）：**蓝色车头 + 黄色车斗**
- 挖挖（挖掘机）：黄色
- 吊吊（吊车）：黄色
- 转转（搅拌车）：绿色车身
- 水水（洒水车）：蓝色
- 铲铲（铲车）：橙色
- 推推（推土机）：黄色或偏土色
- 压压（压路机）：黄色

> 颜色不更改，是第一优先级。

### 2️⃣ 车辆结构保持真实（不拟人化）
- 不添加卡通眼睛或嘴巴
- 不画表情、不拟人化车体
- 不夸张变形（比例接近真实工程车）
- 机械结构真实（轮胎/履带/液压杆/吊臂等不能“简化成玩具”）

### 3️⃣ 车窗与驾驶舱（防止冒出驾驶员）
- 车窗：反光或深色不透明，**看不到车内**
- 禁止：驾驶室内景、透过挡风玻璃看到驾驶舱细节、挡风玻璃特写
- 镜头：优先外景中远景 / 侧面 / 45° 外观

## 二、场景（必须现实）
可用：工地、城市街道、桥梁维修现场、公园施工区、垃圾处理场、建筑现场  
不用：魔法场景、奇幻森林、童话化背景、科幻未来城市

## 三、情绪表达方式（不用“脸”）
情绪只用：
- 动作节奏（快/慢、停一下、回头看）
- 旁白语言（“他停了一会儿”）
- 任务进展（困难→想办法→解决）

## 四、禁止事项（硬规则）
- 不给车加五官（眼睛/嘴巴/眉毛/表情）
- 不让车直接开口说话（用旁白表达）
- 不改变车辆颜色与能力定位
- 不让车飞起来，不出现超现实能力
- **画面中不出现任何人类**（工人/路人/驾驶员/手脚/人影/倒影都不出现）

## 五、推荐关键词（可选）
- realistic construction vehicle illustration
- gentle watercolor style
- realistic structure
- exterior view only
- opaque reflective windows
- no people / no driver / no interior view
